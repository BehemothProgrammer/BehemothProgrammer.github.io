PO'ed EX Tools v1.0.0 Documentation

By BehemothProgrammer

Installation

Extract the games assets from the "POed.kpf" file
  1. Navigate to your PO'ed Definitive Edition install folder.
  2. Unzip the assets from the "POed.kpf" file.
Open Blender and navigate to Edit > Preferences
  1. Click on the Add-ons tab.
  2. Click the Install... button and select the POedEXBlenderTools.zip file you downloaded.
  3. If you don't see the Add-on afterwards type in the search field “poed” to quickly find the add-on Import-Export: POed EX Tools.
  4. Enable the add-on by clicking the checkbox and expand the field to see that there are 2 folder paths you must set before using this add-on.
  5. The PO'ed .exe Folder field must be assigned to the PO'ed Definitive Edition install folder
  6. The Assets Folder field must be assigned to the folder where you extracted the contents of the POed.kpf file. If you haven't done that yet then do that now.
  7. Click the Save Preferences button

Importing and Exporting

  • To import or export levels go to the main toolbar in Blender under File > Import or File > Export.
  • In the View tab it's best to set the Clip Start Distance to 1.0 and Clip End Distance to 100000.0
  • To export your .json level into the game you must launch the game and use the "exportjsonleveltobff" console command.
