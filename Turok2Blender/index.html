This is an addon for Blender that allows you to import and export Staticmeshes, Skinnedmeshes, and Animations from and to Turok 2 EX.
Please read the online documentation for details on installing, importing, exporting, and editing.


Change Log

Version 1.2.0 (2020-04-10)
  • The KFA(Key Frame Action) Offset Point now has a "Show Point On Root Bone" option. Checking this will make the point parented to the Root Bone. Turn this on if the animation will be used with Root Animation on in game. Else turn it off and it will be parented to the model. Model and Root Bone Z location is not used, only XY.
  • The KFA radius value is correct now (argRadius / 256 * 10.24)
  • Added tooltip for the KFA Offset - reminding users that event 11 (Fire Weapon Projectile) does not use the offset value.
  • Added tooltip for the KFA Bone index values - that it's not used at all. Changed the validation check for this from an error to a warning.
  • Actual Damage value in KFA is now displayed beside it.
  • Auto importing an animation with a skinnedmesh will first check for a filename without the _<number> suffix.
  • Removed the open addon folder button from the tool panel
  • For the tool panels staticmesh list, when selecting a mesh it will automatically center the view on the object
  • When importing a material, culling is always set to "use setting". (Except if it's front)
  • Got a better image for the weapon camera background
  • Bounding Box object types that are parented to a bone will now display their bone index and name in the object panel.
  • A new def file in your Turok 2 mod project can now be used for defining custom user events in the Key Frame Action UI. This is for advanced users only. See the file defs/userEvents_example.txt in this add-on's folder for more details.
  • Added Skinnedmesh quick selection list in the tool panel (Warning: Loading a skinnedmesh here will force all actions, materials, and images to be removed from Blenders data)
