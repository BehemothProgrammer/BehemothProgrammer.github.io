This is an addon for Blender that allows you to import and export Staticmeshes, Skinnedmeshes, and Animations from and to Turok 2 EX.
Please read the online documentation for details on installing, importing, exporting, and editing.


Change Log

Version 1.1.0 (2020-04-04)
  • Requires a minimum Blender version of 2.82.7
  • Duplicate vertices are removed when importing static and skinned meshes
  • Turok 2's mirror texture wrapping and reflective material now renders in a similar way inside Blender
  • The armature mesh modifier properties show_in_editmode and show_on_cage are now set to True when importing a skinnedmesh
