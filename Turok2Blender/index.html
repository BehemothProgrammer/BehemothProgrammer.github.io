This is an addon for Blender that allows you to import and export Staticmeshes, Skinnedmeshes, and Animations from and to Turok 2 EX.
Please read the online documentation for details on installing, importing, exporting, and editing.


Change Log

Version 1.5.0 (2020-04-16)
  • Reorganized UI with tabs to make it look better and more compact.
  • Fixed preferences property "Hide Unused Key Frame Action Args" from also turning on/off the "Auto name Animations" property.
  • Material cull property now effects the blender setting use_backface_culling property if you set it to Back or None.
  • Setting the Blend or AlphaTest property now effects blenders settings blend_method property.
  • Setting Full Bright now links the texture color to the emission color.
  • The Reflection and Mirror UV Shading Nodes get removed when they are unused.
