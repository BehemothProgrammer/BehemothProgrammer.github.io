This is an addon for Blender that allows you to import and export Staticmeshes, Skinnedmeshes, and Animations from and to Turok 2 EX.
Please read the online documentation for details on installing, importing, exporting, and editing.


Change Log

Version 1.4.0 (2020-04-15)
  • Fixed overlapping triangles not being imported. Blender wouldn't allow me to create any face that overlapped another even if the winding order was different. I found a way around it, so now triangles that overlap and have a different winding order and/or different material will be imported.
  • Mesh validation now checks if any material param name is empty and the material param UI is now red when theres no name.
  • There's a new setting in preferences to hide Key Frame Action Args that are unused.
  • Rearranged Action Key Frame UI
  • Moved the example def file that was in this addons "defs/userEvent_example.txt" to "examples/userEvents.txt"
  • A new def file in your Turok 2 mod project can now be used for defining custom Hot Points in any hot point property UI. This is for advanced users only. See the file "examples/hotPoints.txt" in this add-on's folder for more details.
