Installation

Important: These tools were only tested on version 2.82 of Blender. If you are having problems in later versions download 2.82

Make sure to unzip the contents of ShadowManEX01.kpf, ShadowManEX02.kpf, ShadowManEX03.kpf, ShadowManEX04.kpf in your steam/gog/epic store/other "Shadow Man Remastered" Folder before continuing.

Open Blender and navigate to Edit > Preferences
  1. Click on the Add-ons tab.
  2. Click the Install... button and select the ShadowManEXBlenderTools_2.0.0.zip file you downloaded.
  3. If you don't see the Add-on afterwards type in the search field shadow‚Äù to quickly find the add-on Import-Export: Shadow Man EX Tools.
  4. Enable the add-on by clicking the checkbox and expand the field to see that there are 2 folder paths you must set before using this add-on.
  5. Assign the Shadow Man Folder field to where the thoth_x64.exe is located. The Res 1 Folder field should be assigned to the folder where you unzipped the contents of ShadowManEX01.kpf. Do the same for the other res folder paths.
  6. Click the Save Preferences button
  7. Click the Reload All Defs to load the def files that come with this addon. Once the files are loaded there will be a checkmark beside the button indicating that they were loaded successfully. If you edit any def files you can click this button to reload them. This button is also available in the 3DView Tool Panels Misc Tab.

Additional Info

If you see this icon next to a property it means that when you set the property all selected objects will automatically have there property set to the same value as well. And right clicking and selecting 'Copy to Selected' will do nothing. In order to set other properties for all selected objects you must do it the normal Blender way of right clicking on the property and selecting ''Copy to Selected'
