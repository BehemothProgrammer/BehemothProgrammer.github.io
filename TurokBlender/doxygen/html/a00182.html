Detailed Description

A map/dictionary that stores key/values as kStr. Example code to iterate each key/value:

kScriptTMap tmap;
kStr key, value;
tmap.ItBegin();
while(tmap.ItNext(key, value))
{
Sys.Print("Key: " + key + ", Value: " + value);
}
